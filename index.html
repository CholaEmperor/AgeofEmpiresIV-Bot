Age of Empires IV Discord Bot Guide

Commands

PlayerID Management

Description: Base command for PlayerID management.

Usage: [p]aoe4

Admin Commands

Description: Base command for Admin.

Usage: [p]aoe4 admin

Admin Commands:

add

Description: Used to add player ID.

Usage: [p]aoe4 admin add <player_id> [user_id]

set

Description: Used to manage announcement channel.

Usage: [p]aoe4 admin set
